# Cowbell Teleport
A RuneLite plugin that replaces all teleport animations with the cowbell amulet teleport animation for a fun visual effect!

## Features
- Automatically replace standard teleport animations (spell teleports, item teleports, tabs) with the cowbell amulet animation
- Toggle the feature on/off via the plugin configuration
- Customizable animation ID for the cowbell replacement animation
- Works with all teleport types: Modern spellbook, Ancient spellbook, Arceuus spellbook, Lunar spellbook, jewelry, and teleport tabs

## Configuration
- **Enable Cowbell Teleport**: Toggle to enable/disable the animation replacement
- **Cowbell Animation ID**: Customize which animation ID is used for the replacement (default: 8939)

## How to Use
1. Install the plugin in RuneLite
2. Open the plugin settings
3. Enable "Enable Cowbell Teleport" toggle
4. Enjoy your new teleport animations!

## Supported Teleports
This plugin replaces animations for:
- Modern Spellbook: Home, Varrock, Lumbridge, Falador, Camelot, Seers', Ardougne, Watch Tower
- Ancient Spellbook: All teleport variations
- Arceuus Spellbook: All teleport variations
- Lunar Spellbook: All teleport variations
- Jewelry teleports: Rings, amulets, and other jewelry
- Teleport tabs
